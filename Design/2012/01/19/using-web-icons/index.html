Using the Web Icons

Jan 19, 2012
Design
Nesterov Vitaliy

This template uses Typicons web font, provided by Fontello font bundling service. It allows you to quickly add nice icons into your pages by using css tags.

To add an icon somewhere in the template simply do:

<i class="icon-home"></i>
